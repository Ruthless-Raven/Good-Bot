# Good-Bot

Good-Bot is a Python module for dealing with word pluralization.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install training
```

## Usage

```python
import training

# returns 'words'


# returns 'geese'


# returns 'phenomenon'

```
